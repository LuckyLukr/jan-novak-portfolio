# Jan Novák — Vývojářské portfolio

Vytvořte vývojářské portfolio SPA: fixní postranní navigace vlevo s avatarem, jménem, odkazy (O mně, Dovednosti, Projekty, Kontakt) a sociálními ikonami dole. Hlavní oblast: sekce O mně s bio, Dovednosti s progress bary nebo tagy, mřížka projektů (4–6 karet s obrázkem, názvem, popisem, tech tagy a odkazy), a Kontakt s formulářem (jméno, email, zpráva). Patička s copyrightem. Použijte čisté tmavé/světlé téma.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Tailwind CSS is preconfigured through Vite and loaded from `src/tailwind.css`.

Built with React + Vite. Deployed via Jean AI.
